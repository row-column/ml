 全国 第二届 Java 技术 及其 应用 学术 交流会 征文 
 　 　 自 1998 年 6 月底 成功 地 召开 了 全国 第一届 Java 技术 及其 应用 学术 交流会 以来 ， Java 技术 又 有 了 惊人 的 发展 ： 推出 了 功能 大为 扩充 、 速度 也 有 较大 提高 的 Java   2 ； 涉及 到 千家万户 的 、 并 将 对 分布式计算 处理 及其 应用 产生 深远影响 的 Jini 技术 ； 基于 Java 的 嵌入式 技术 的 流行 ； 各行各业 已 涌现 一批 基于 Java 技术 的 信息处理 应用 系统 ， 如用 Java 技术 实现 的 电子 商贸 、 远程教学 、 远程 购物 、 远程 医疗 、 网上银行 等 一批 应用 成果 。 中国计算机学会 、 中国 自动化 学会 计算机 应用 委员会 、 中国 电子 学会 计算机 工程 与 应用 委员会 、 电子部 微型机 信息网 等 学会 与 组织 将 于 1999 年 8 月 联合 召开 “ 全国 第二届 Java 技术 及其 应用 学术 交流会 ” 。 
 　 　 征文 重点 及 范围 ： 国内外 Java 技术 及其 应用 发展 综述 及 专题报告 和 专题讲座 ； Java 技术 产品 ( 包括 构件 、 模块 及 应用软件 ) 开发 及 应用 ； Java 嵌入式 计算机 及 家用 信息 产品开发 与 应用 成果 ； Java 计算 、 平台 、 开发工具 及其 应用 经验 ； Java 技术 在 电子商务 、 网络通信 、 企业 网络 、 MIS 、 OA 中 的 应用 ； Java 在 各 行业 的 应用 成果 、 经验 及 典型 实例 ； 我国 Java 技术 发展 战略 及 政策措施 建议 。
